# Módulo de Diseño

Este módulo contiene todos los **artefactos de diseño** del sistema, incluyendo diagramas, prototipos y documentación visual.

## Contenido

- [**Documentación completa de diseño**](./design_documentation.md): Guía completa sobre arquitectura, diagramas de clases, despliegue y trazabilidad.
- [**Diagramas**](./diagrams/): Diagramas del sistema (dominio, casos de uso, secuencia, C4).
- [**Trabajos prácticos**](../practical_work/): Documentación detallada de los trabajos prácticos realizados.

## Diagramas

### Diagramas fuente (`.drawio`)
- [**Diagrama de dominio**](./diagrams/domain_diagram.drawio): Estructura conceptual de las entidades principales del sistema.
- [**Diagrama de casos de uso**](./diagrams/use_cases_diagram.drawio): Visualización de las funcionalidades del sistema desde la perspectiva del usuario.
- [**Diagrama de secuencia**](./diagrams/sequence_diagram.drawio): Interacciones temporales entre objetos del sistema.
- [**Diagramas C4**](./diagrams/c4/C4.drawio): Modelo de arquitectura en cuatro niveles (Contexto, Contenedores, Componentes, Código).
