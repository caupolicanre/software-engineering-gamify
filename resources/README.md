# Recursos (Resources)

Este módulo incluye todos los recursos auxiliares del proyecto, como imágenes, íconos y otros elementos utilizados en la documentación y el diseño.

## Estructura

```
resources/
├── images/             # Imágenes y recursos visuales
└── README.md           # Este archivo
```

## Uso

Los recursos de este directorio son utilizados por:

- Documentación en `docs/`
- Interfaces de usuario
- Assets del proyecto

Para agregar nuevos recursos, por favor mantén una estructura organizada y nombres descriptivos.
